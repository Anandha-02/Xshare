# Xshare Frontend

This is the **frontend** for the Xshare app, built with **React + Vite** (or **Next.js**).

## Features

- User Authentication (Login/Register)
- File Upload / Download
- Progress bar and chunked uploads
- Responsive and modern UI
- JWT token authentication

## Installation

```bash
# Clone the repo
git clone https://github.com/yourusername/xshare-frontend.git
cd xshare-frontend

# Install dependencies
npm install
