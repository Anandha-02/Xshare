# Xshare - File Transfer Application

Xshare is a full-stack file sharing application that allows users to **upload, download, and share files** securely between devices. It consists of a **Node.js + Express backend** and a **React + Vite frontend**.

---

## **Features**

- User authentication (Register/Login)
- Secure file upload and download
- File list management
- Large file support with upload progress
- Responsive and modern UI
- JWT-based authentication
- Docker-ready for easy deployment
